Comments

ในการรันโปรแกรม Interpreter จะไม่ประมวลผลข้อความตัวอักษรที่อยู่ใน Comment

Comment ใช้ทำอะใน Python

  • ใช้อธิบายโค้ด ให้กับตนเอง และผู้อื่น
  • ทำให้โค้ดอ่านง่ายขึ้น
  • ปิดการทำงานบางส่วนของโค้ต

รูปแบบการ Comment ในภาษา Python

  1. การคอมเมนต์แบบบรรทัดเดียว ตัวอักษรที่อยู่หลังเครื่องหมาย # ในบรรทัดนั้นจะกลายเป็นคอมเมนต์
  2. การคอมเมนต์แบบหลายบรรทัด """ This is a comment """ ตัวอักษรที่อยู่ภายใต้ double quote จะเป็นคอมเมนต์ทั้งหมด
