Variables

Variable หรือ ตัวแปร เปรียบเทียบได้ว่า ตัวแปรนั้นคือ "กล่อง" ที่ภายในกล่องจะมีข้อมูลเก็บไว้อยู่


การสร้างตัวแปร

ชื่อตัวแปร ตามด้วยเครื่องหมายเท่ากับ ( = ) หลังเครื่องหมายเท่ากับจะเป็นข้อมูลของตัวแปร


การตั้งชื่อตัวแปร

  • ตัวแปรจะต้องขึ้นต้นด้วยตัวอักษรหรือ underscore ( _ )
  • ตัวแปรไม่สามารถขึ้นด้วยตัวเลขได้
  • ตัวแปรมีได้แค่ ตัวอักษร ตัวเลข และ underscore (A-z, 0-9, _)
  • ตัวอักษรพิมพ์ใหญ่ และตัวอักษรพิมพ์เล็ก ( var, Var, VAR ทั้งสามตัวนี้เป็นตัวแปรคนล่ะตัวกัน )
