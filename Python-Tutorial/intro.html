ทำความรู้จักกับ Python

Python คืออะไร

Python เป็นภาษาโปรแกรมมิ่งที่ได้รับความนิยมสูงเป็นอันดับ 1 ในปัจจุบัน ( 2022 )


Python ใช้ทำอะไรได้บ้าง

  • ใช้สร้าง application
  • ใช้สร้างเกม โดยใข้ framework เช่น PyGame
  • ใข้เชื่อมต่อกับระบบฐานข้อมูล (data base)
  • ใช้จัดการกับข้อมูลขนาดใหญ่ได้ และคำณวนคณิตศาสตร์ที่ซับซ้อนได้

ทำไมต้องใช้ Python

  • สามารถรันได้บนหลายแพลตฟอร์ม (Windows, Mac, Linux, Raspberry Pi, etc)
  • มีไวยากรณ์ หรือ รูปแบบภาษา ที่คล้ายกับภาษาอังกฤษ ทำให้โค้ดสามารถอ่านเข้าใจได้ง่าย
  • รันบนรูปแบบ interpreter (อ่านโค้ดทีล่ะบรรทัดเมื่อกดรัน) ทำให้โปรแกรมสามารถรันได้ทันที
